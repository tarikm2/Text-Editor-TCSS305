This project represents a Text-editor application developed with the Java.Swing Librarys.
This Text editor will include file open, save, and new file creation functionality as 
Well as textwrap, font size, font style, font type, and help menue functionality 
when complete. basic GUI components are currently implemented along with 
functionality for font type and text wrap functionality. All other functionality
currently under development.
